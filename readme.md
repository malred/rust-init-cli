# rust-vite-cli

命令行快速创建vite项目

# 使用

1. 下载 或 git clone 该库

2. 编译 或 将根目录的release文件夹放到想要放的地方

我编译的环境是win10 x86 intel

> 想要全局使用请配置环境变量
![img.png](doc/img.png)

3. 在命令行中运行

```shell
rust-vite-cli create
> your project name?
test
> select a framework:
> react
> vue
vue
> select a variant:
> typescript(ts)
> javascript(js)
js
```
